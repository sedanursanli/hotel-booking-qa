# QA Portfolio – Restful Booker Platform Demo

Bu proje, `https://automationintesting.online/` platformu üzerinde gerçekleştirilen manuel test çalışmalarını içermektedir.

Amaç; gerçek bir QA sürecini simüle ederek test planlama, test senaryosu oluşturma, test case yazımı, bug reporting ve smoke testing süreçlerini dokümante etmektir.

---

# Project Scope

Test çalışmaları aşağıdaki modüller üzerinde gerçekleştirilmiştir:

- Booking
- Admin Panel
- Contact Form
- UI/UX Navigation

---

# Test Approach

Projede aşağıdaki test yaklaşımları kullanılmıştır:

- Manual Testing
- Functional Testing
- Negative Testing
- Exploratory Testing
- Smoke Testing
- UI/UX Testing

# Bugs Identified

Test execution sürecinde aşağıdaki örnek hatalar tespit edilmiştir:

- Geçmiş tarih ile rezervasyon oluşturulabilmesi
- Aynı oda numarası ile duplicate oda oluşturulabilmesi
- Quick Links bağlantılarının çalışmaması
- Phone alanının harf kabul etmesi
- Sosyal medya yönlendirmelerinin çalışmaması
